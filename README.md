## Amigo Secreto
Desenvolvimento de uma aplicação que permita aos usuários inserir nomes de amigos em uma lista para, em seguida, realizar um sorteio aleatório e determinar quem é o "amigo secreto".

## Funcionalidades
- adicionar nomes
- validar entrada
- visualizar a lista
- sorteio aleatório
